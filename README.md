# PAC-MAN
Project for OOPL course, 2024 Spring, NTUT

# 遊戲簡介

- 遊戲名稱: Pac Man

- 操作方式: 使用方向鍵操作，角色會自動前進，遇到牆壁會停下

# 任務概述

- 控制遊戲的主角小精靈吃掉藏在迷宮內所有的豆子，並且不能被鬼魂抓到 


# 預期工作內容&開發進度
- 開始畫面
- 地圖
- 人物操作
- 敵對NPC移動
- 死亡判定
- 死亡頁面
- 能量球(反吃鬼魂)
- 反吃機制
- 豆子生成
- 過關判定
- 水果(加分機制)生成
- 分數系統
- 關卡增加
- 實習報告
